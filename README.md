# React com SQL (CRUD)

## Installation

### Client

  1. `$ cd client`
  2. `$ npm install` or `$ yarn install`
  3. `$ npm start` or `$ yarn start`
  
### Server

  1. Change your mySQL database data `server/index.js`
  2. `$ cd server`
  3. `$ npm install` or `$ yarn`
  4. `$ npm rum devStart`
  
## Functionalities

  + Create books in the database
  + Read books in the database
  + Update books in the database
  + Delete books in the database
  + Click card to edit and delete
